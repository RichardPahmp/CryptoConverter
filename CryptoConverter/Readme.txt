Klienten startas med main-funktionen i klassen MainController.
Servern startas med main-funktionen i klassen ServerMain.

För att kunna köra projektet krävs en JDK 8+ och pluginen "e(fx)clipse" installerad i eclipse.
För att ändra inställningar för vilken databas som ska användas så ändrar man i files/DatabaseConfig.properties.
För att ändra vilken port servern lyssnar på ändrar man i files/server.properties
För att ändra vilken ip-address och port klienten kopplar upp sig till ändrar man i files/ClientSettings
